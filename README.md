## ✏️ Modifier depuis téléphone

Tu peux éditer les fichiers directement depuis ton téléphone sans avoir besoin de terminal :

1. **Ouvrir le fichier** : Va sur GitHub, clique sur le fichier que tu veux modifier (ex: `index.html`)
2. **Cliquer le crayon** : Appuie sur l'icône ✏️ (Edit this file)
3. **Modifier le code** : Change le texte ou le code
4. **Sauvegarder** : Scroll vers le bas, écris un message (ex: "Modifie le titre"), puis **"Commit changes"**
5. **Et voilà !** Le site se met à jour automatiquement 🎉
